# 🏥 Healthcare Client Analytics Dashboard

## 📌 Overview

This project analyzes client service data in a healthcare environment to evaluate performance, monitor trends, and generate actionable insights to improve client satisfaction and operational efficiency.

---

## 🎯 Objectives

* Analyze client engagement and session trends
* Evaluate satisfaction levels and response times
* Compare service performance across different service types
* Identify areas for improvement in client services

---

## 📊 Key Metrics

* Total Clients: 2.58K
* Average Satisfaction Score: 3.17
* Average Response Time: 31 minutes
* Total Sessions: 63 in 2026

---

## 📈 Dashboard Features

* Sessions Trend Over Time
* Response Time by Issue Type
* Clients Distribution by Country
* Service Type Performance

## 🚀 Key Insights

### 📉 Sessions Trend

* Sessions peaked in August 2025 (1179 sessions), a consistent decline followed over the next months, by January 2026, sessions dropped significantly to only 63. This sharp decrease indicates a major drop in client engagement

---

### ⏱️ Response Time Analysis

* Stress-related issues have the highest response time (33.45 minutes), other issue types (Work, Anxiety, Family) show similar response times. Overall variation is minimal, but stress cases may require faster handling

---

### 🌍 Clients Distribution

* Clients are evenly distributed across countries:

  * Indonesia: 25.6%
  * USA: 25.4%
  * UK: 25.1%
  * UAE: 23.9%

* This indicates a balanced global client base

---

### 🧠 Service Performance

* Support service has the highest usage (3766 sessions). Therapy and Coaching show slightly lower usage, indicates strong demand for support services, other services may require improvement or better promotion
